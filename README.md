# GaleriaLocal

App móvil con Expo y React Native.

## Ejecutar en móvil

**Prerequisitos:** Node.js y la app Expo Go en tu teléfono.

1. Instala dependencias:
   npm install
2. Inicia Expo:
   npm run dev
3. Abre Expo Go en tu teléfono y escanea el QR que aparece en la terminal.

## Notas

- No usa mocks para los módulos de Expo.
- Está pensado para ejecutarse en Android/iPhone, no como app web.
